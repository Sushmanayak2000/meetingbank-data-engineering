# Download MeetingBank Dataset

The MeetingBank dataset (~250 MB) is **not included** in this repository due to size limitations.  
Please follow these steps to download it:

1. Go to the Zenodo page: [MeetingBank Dataset](https://zenodo.org/records/7989108)
